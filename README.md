# marvel_champions_bgg_export

Export Marvel Champions board game plays from BoardGameGeek (BGG) and create statistical visualizations.

## Overview

This project consists of two main scripts that work together to fetch, analyze, and visualize Marvel Champions board game play data from BoardGameGeek.

---

## BGG_Export.py

### Functionality

This script fetches all Marvel Champions plays for a specific BGG user and generates multiple CSV files containing detailed statistics.

### Features

- **Data Fetching**: Connects to the BoardGameGeek XML API to retrieve all play records for Marvel Champions
- **Comment Parsing**: Extracts structured game information from play comments using a pattern: `<Hero> vs <Scenario> - <Result>`
- **Hero Tracking**: Identifies and counts which heroes were played
- **Scenario Analysis**: Categorizes plays by scenario and modular content
- **Aspect Statistics**: Tracks which card aspects (Aggression, Justice, Leadership, Protection, etc.) were used
- **Hero-Aspect Combinations**: Records which aspect was paired with each hero for every play
- **Win/Loss Tracking**: Records game outcomes

### Configuration

- `USERNAME`: BGG username to fetch plays for (default: "Almecho")
- `GAME_ID`: BoardGameGeek ID for Marvel Champions (285774)
- `HEROES`: List of all playable heroes
- `ASPECTS`: List of card aspects (Aggression, Justice, Leadership, Protection, Basic, 'Pool, Precon)
- `SCENARIOS`: List of all game scenarios
- `MODULARS`: List of modular encounter sets

### Output Files

1. **marvel_champions_plays.csv**: Complete play history with all extracted data
   - Columns: id, date, quantity, length, game_name, hero, scenario, result, and individual scenario/modular markers

2. **marvel_champions_scenario_stats.csv**: Scenario play counts
   - Columns: scenario, count (sorted by frequency)

3. **marvel_champions_modular_stats.csv**: Modular encounter set play counts
   - Columns: modular, count (sorted by frequency)

4. **heroes_total.csv**: Total plays per hero
   - Columns: Hero, Count

5. **heroes_aspects.csv**: Hero and aspect combinations
   - Columns: Hero, Aspect, Count

6. **marvel_champions_aspect_stats.csv**: Overall aspect usage statistics
   - Columns: aspect, count

## Visualize.py

### Functionality

This script reads the CSV files generated by BGG_Export.py and creates a comprehensive visualization dashboard with six different chart types.

### Features

- **Multiple Chart Types**: Creates a 3x2 grid of visualizations
- **Themed Styling**: Uses Marvel-themed colors (red, black, white, gold)
- **Aspect-Specific Colors**: Custom colors for each card aspect
- **Interactive Visualization**: Uses matplotlib and seaborn for professional charts

### Charts Generated

1. **Top 10 Most Played Heroes** (Bar Chart)
   - Shows which heroes have been played most frequently
   - Sorted in descending order

2. **Aspect Distribution** (Donut Chart)
   - Displays the proportion of each card aspect used
   - Shows percentages and aspect labels
   - Uses themed colors from the Marvel Champions game

3. **Top 10 Most Played Scenarios** (Bar Chart)
   - Ranks scenarios by number of times they were played
   - Helps identify favorite or frequently-played villains

4. **Overall Win Rate** (Pie Chart)
   - Shows the win/loss ratio across all plays
   - Displays percentages for games won vs. lost
   - Green for wins, red for losses

5. **Top 5 Heroes - Aspect Preferences** (Stacked Bar Chart)
   - Shows which aspects were used with each of the top 5 heroes
   - Allows comparison of aspect distribution across heroes
   - Includes legend for aspect colors

6. **Game Activity Over Time** (Line Chart)
   - Monthly play frequency trend
   - Shows how gaming activity varies over time
   - Helps identify periods of high or low activity

### Input Files

Expects the following CSV files (generated by BGG_Export.py):
- heroes_total.csv
- marvel_champions_aspect_stats.csv
- marvel_champions_scenario_stats.csv
- marvel_champions_plays.csv
- heroes_aspects.csv

### Styling

- Uses seaborn whitegrid style for clean appearance
- Font size: 10pt (adjustable)
- Figure size: 16x20 inches
- Marvel-themed color palette with custom aspect colors

## Usage

1. **Generate Data**:
   ```
   python BGG_Export.py
   ```
   This fetches all plays from BGG and creates the CSV statistics files.

2. **Create Visualizations**:
   ```
   python Visualize.py
   ```
   This reads the CSV files and displays the visualization dashboard.

## Dependencies

- `requests`: For BGG API calls
- `pandas`: For data manipulation and CSV handling
- `matplotlib`: For creating charts
- `seaborn`: For enhanced visualization styling
